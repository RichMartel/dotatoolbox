About the site

I created this site as an outlet for new apps and tools that use statistics to help dota players get more from their game. Dota 2 has such a large user base (600,000+ people playing at any one time, and 12,000,000+ unique players in a month), that creating any useful app for the game is sure to be valuable.

About me
