Dota 2 Hero Builds

Build helper


What is this?

  • This is an automatic item build creator for the game Dota 2
  • It uses statistics (popularity, winrates, etc.) to create the item build listings
  • There are 3 builds shown for each hero: "All skill" (any MMR), "High skill" (> 3800 MMR), "Pro skill" (tournaments)
  • It budgets the hero\'s ability to buy items based on the average GPM for that hero, at that skill, by the 35 minute mark

How to use?

  • Click on a hero at the top
  • Suggested build items are under "Build"
  • Possible early item options are under "Early options"
  • Possible late item options are under "Late options"
  • Item tooltips will give you the following data: itemname (cost) | WR: winrate | BR: buildrate
  • Right click any item and select "Open link in a new tab" to look up the DotaBuff information for that item
